  • RNA Sequencing and Normalization
    • RNA Sequencing
      • read mapping and count data
        • count r_ik is the number of reads that map on trascript i in the sample k (raw counts)
        • probability that 1 fragment is mapped is : p_ik = [(T_ik*L_i)/l] / [sum(T_jk*Lj)/l]
        • expected value of r_ik is : E[r_ik] = n_k * p_ik
        • Gene expression measured in RPKM : Expr_ik = (r_ik*10^6) / (n_k*L_i).
          • RPKM is Reads Per Kilobase of transcript per Million mapped reads
    • Normalization
      • Measures can be affected by random errors, that can be reduced by replicated measurements, and by systematic error, that can be reduced by normalization.

        • Quantile normalization
          1. Put each column in ascending order in the matrix Data.sorted

  • Differentially Expression Analysis - Selection Classical Tests
    • selection of differentially expressed genes
    • Student t-test comparison
      • y_obs = (m_a-m_b) / [s_p*sqrt(1/n_a + 1/n_b)]
      • s_p^2 = [(n_a -1)*s_a^2 + (n_b -1)*s_b^2]/(n_a+n_b-2)
      • df = n_a + n+b -2
      • then we either compare y_obs with a confidence threshold (theta) or p_value with alfa
    • Powerful variants of Student t-test
      • Welch-Student t-test
      • Wilcoxon test
        • rank sum statistic
        • procedure
          1. N = n_a + n_b
          2. Sort the expression values in ascending order and assign a rank from 1 to N
          3. If two values are identical, I assign them as rank the mean of the ranks
          4. pick the group with less cardinality e.g let n_a : n_a <= n_b, and calculate R_obs
            • R_obs = [R_a-(n_a(N+1))/2]/[sqrt((n_a*n_b*(N+1))/12)]
          5. then we reject or not our null hypothesis comparing R_obs with theta or p-value with alfa
      • Permutation test

    • Biological annotation

    • Functional Enrichment test
      • list of all genes analysed and subset of selected as differentially expressed can be annotated with their relevant functional information.
      • then we can ask whether the number of selected genes is significantly greater (enriched) than what I would have obtained with a random selection of genes
        • Enrichment analysis is the verification of this hypothesis, a statistical test widely used for this is the Fisher Exact Test
        • Fisher Exact test
          • p represents exactly the probability of randomly observing the values when the things are fixed
      • GSEA (Gene Set Enrichment Analysis)
        • goal is to identify functional groups of genes characterized by one coherent differential expression
        • For this GSEA we analyse the ranking of statistics y used for selection without setting a threshold on y
        • Procedure
          1. Order the N genes according to the chosen y statistic
          2. We then calculate the running sum RS as a function of the rank k of the statistic y: RS(k)=sum[|y_i|^p/sum(|y_j|^p)] - sum (1/(N-N_set))
          3. default for p is 1
          4. the maximum deviation from 0 of RS(k) is called Enrichment Score (ES)
